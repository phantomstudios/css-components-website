### Adding variants

You can add variants by using the `variants` config object key. There is no limit to how many variants you can add.

<Preview>
  <div style={{ display: "flex", gap: "20px" }}>
    <DemoButton color="teal">Teal Button</DemoButton>
    <DemoButton color="gray">Gray Button</DemoButton>
  </div>
</Preview>

```jsx
const Button = styled("button", {
  css: "baseButton",
  variants: {
    color: {
      teal: "colorTeal",
      gray: "colorGray",
    },
  },
});

() => <Button color="teal">Button</Button>;
```

A variant accepts the same css style object as the [base styles](/docs/composing-components).

### Multiple variants

<Preview>
  <div style={{ display: "flex", gap: "20px", alignItems: "center" }}>
    <DemoButton color="gray" size="small" />
    <DemoButton color="teal" size="small" />
    <DemoButton color="gray" size="large" />
    <DemoButton color="teal" size="large" />
  </div>
</Preview>

```jsx
const Button = styled("button", {
  css: "baseButton",
  variants: {
    color: {
      teal: "colorTeal",
      gray: "colorGray",
    },
    size: {
      small: "sizeSmall",
      large: "sizeLarge",
    },
  },
});

() => (
  <Button color="teal" size="large">
    Button
  </Button>
);
```

### Boolean variants

Variants can be booleans by using `true` as the key.

<Preview>
  <DemoButton outlined />
</Preview>

```jsx
const Button = styled("button", {
  // base styles
  css: "baseButton",
  variants: {
    outlined: {
      true: "outlined",
    },
  },
});

() => <Button outlined>Button</Button>;
```

### Compound variants

In the case of needing to set styles of a variant, based on a combination of other variants, you can use the `compoundVariants` feature.

<Preview>
  <div style={{ display: "flex", gap: "20px", alignItems: "center" }}>
    <DemoButton color="teal" outlined />
    <DemoButton color="gray" outlined />
  </div>
</Preview>

```jsx line=13-35,38
const Button = styled("button", {
  css: "baseButton",
  variants: {
    color: {
      teal: "colorTeal",
      gray: "colorGray",
    },
    outlined: {
      true: "outlined",
    },
  },

  compoundVariants: [
    {
      color: "teal",
      outlined: true,
      css: "colorOutlinedTeal",
    },
    {
      color: "gray",
      outlined: true,
      css: "colorOutlinedGray",
    },
  ],
});

() => (
  <Button color="teal" outlined>
    Button
  </Button>
);
```

### Default variants

You can use the `defaultVariants` feature to set a variant by default:

<Preview>
  <div style={{ display: "flex", gap: "20px", alignItems: "center" }}>
    <DemoButton />
  </div>
</Preview>

```jsx
const Button = styled("button", {
  css: "baseButton",
  variants: {
    color: {
      teal: "colorTeal",
      gray: "colorGray",
    },
  },

  defaultVariants: {
    color: "teal",
  },
});

() => <Button>Button</Button>;
```
