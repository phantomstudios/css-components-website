### Install CSS Components

Install CSS Components from your terminal via npm or yarn.

```bash
# With npm
npm install @phntms/css-components

# With yarn
yarn add @phntms/css-components
```

### Import it

Import `styled` from `@phntms/css-components`.

```tsx
import { styled } from "@phntms/css-components";
```

### Use it

Use the `styled` function to create a component and add styles to it.

```tsx
import { styled } from "@phntms/css-components";

const Button = styled("button", {
  css: "root",
});
```

### Render it

Finally, render the component.

<Preview>
  <DemoButton>Button</DemoButton>
</Preview>

```tsx
import { styled } from '@phntms/css-components';

const Button = styled('button', {...});

() => <Button>Button</Button>;
```
